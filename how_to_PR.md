# How to contribute

Suggestions, paper updates, and pull requests are welcome. Edit `README.md` and follow the format below.

## Entry template

```markdown
* **Short Title**: "Full Title", **`CONFERENCE YEAR`**. [[Paper](link)] [[Website](link)] [[Code](link)]
```

## Link guidelines

- Prefer **arXiv** links over mirrors on other hosts when the paper is on arXiv.
- Link to the **abstract page** (e.g. `https://arxiv.org/abs/...`), not the raw PDF.
